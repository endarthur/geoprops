### Package Installation
The package is installed in **editable/development mode** using:
```bash
pip install -e .
```

This means changes to the source code are immediately reflected without reinstalling.

## Usage

### Running Python Commands
Use `conda run -n geoprops` to run commands in the environment:

```bash
# Run Python script
conda run -n geoprops python script.py

# Run tests
conda run -n geoprops pytest

# Run specific test file
conda run -n geoprops pytest tests/unit/test_basic_imports.py -v

# Interactive Python
conda run -n geoprops python
```

### Development Workflow
```bash
# Make changes to geoprops source code
# Changes are immediately available (editable install)

# Run tests to verify
conda run -n geoprops pytest

# Run specific tests
conda run -n geoprops pytest tests/unit/test_mesh.py -v -k test_mesh_volume
```

### Adding Dependencies
```bash
# Add to pyproject.toml first, then:
conda run -n geoprops pip install package_name

# Or install from pyproject.toml
conda run -n geoprops pip install -e .
```

## Quick Verification
```bash
# Verify installation
conda run -n geoprops python -c "import geoprops as gp; print(gp.__version__)"

# Run all tests
conda run -n geoprops pytest -v

# With coverage
conda run -n geoprops pytest --cov=geoprops --cov-report=term-missing
```
